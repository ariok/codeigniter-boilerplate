Codeigniter-Boilerplate

Easy structure builder

  • Codeigniter-Boilerplate helps you start immediately a new project thanks to a
    simple template based on HTML5 Boilerplate. This guarante the best
    crossbrowser experience and fast/easy CSS styling.
  • Codeigniter-Boilerplate has a prebuild structure which encapsulates single page
    views into a container view which shares parts of code as header, footer and script.
  • Codeigniter-Boilerplate has the primarly aim to help developer building simple
    pages in the shortest time but without rigid structures, that means: allowing
    definition of custom elements for any pages without resctrictions.
    Developer as fast access to title, description, javascript, css, Google Font
    and (obviously) content for any single page.

Technical notes

  • Codeigniter Version: 2.0.2
  • JQuery Version: 1.5.1 (waiting for HTML5 BP update)
  • HTML5 BP Version: 1.0 Stripped

CI Configurations

Codeigniter configuration for Boilerplate starts with:

  • Autoload: url helper
  • Routes: default_controller = "homepage"
  • Index.php: subclass_prefix = "BP_"
  • Base url: http://localhost/codeigniter-boilerplate
